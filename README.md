# Extranet Api Express Backend

[![Express Logo](https://i.cloudup.com/zfY6lL7eFa-3000x3000.png)](http://expressjs.com/)

Fast, unopinionated, minimalist web framework for [node](http://nodejs.org).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# Run
$ npm start
```

```bash
# Watch mode
$ npm run start:dev
```

## Build the app

```bash
# Build
$ npm run-script build
```
